Development of a System of Indicators for a Resource Efficient Europe

Details

Status Finished
Role WP leader (Compilation of EE-MRIO tables)
Duration Sept 2012 - Feb 2016
Funding EU fp7 - Grant Nr. 308552
Website

DESIRE was a FP7 project that aimed to develop and apply an optimal set of indicators to monitor European progress towards resource-efficiency. Within this project, we compiled version 3 of EXIOBASE.
